Compute a list of common constraints for splitting on parent.

Parameters
parenta BranchNode that is currently being split on.
Returns
a vector of EndsDir objects corresponding to the branchiing constraints for parent and all its ancestors. When splitting a problem, we need to compute branch tours for both its children, so we need a list of branch constraints for both of them. This method will return the common branch constraints for the parent node, reserving an extra space in the vector for the current node. Thus if parent has child nodes M and N, to compute a pair of branch tours for M and N you would call this function on parent, and then push/pop the constraints for the individual children.

void CMR::ABC::BranchTourFind::prune_edges ( )

Prune edges which were added by branch tours.

To avoid the core edge set becoming extremely large, this function can be called to delete all edges from BranchTourFind::core_graph which are marked in BranchTourFind::tour_edge_tracker as having been added in computing a branch tour.

Before doing this, it will mark all edges in the best tour as core graph edges, so as not to accidentally delete edges that helped find an augmenting tour.
