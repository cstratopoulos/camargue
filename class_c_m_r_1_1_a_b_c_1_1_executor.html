Get a tour satisfying the branching in edge_stats.

Parameters
[in]edge_statsa vector of edges being branched upon, together with the value they are clamped to.
[in]start_tour_nodesthe tour being used as the starting cycle for computing tour.
[out]found_tourset to true iff a compliant tour was found.
[out]feasset to false iff the branch instructions were proven infeasible.
[out]tourthe branching tour.
[out]tour_valthe length of tour.
Returns
Uses a short run of chained Lin-Kernighan to compute a tour which complies with the branching directions specified in edge_stats, with start_tour_nodes as the starting cycle for the LK run. The resulting tour can be used as a starting basis for cutting and pivoting on a branching problem which complies with edge_stats. The computed tour will be verified for compliance before it is returned. If no compliant tour is found, found_tour will be set to false. If it is proved that edge_stats make it impossible for any tour to be found, feas will be set to false, as will found_tour. If either feas or found_tour is set to false, tour will be cleared and tour_val will be set to a large positive value.

std::vector<int> CMR::ABC::Executor::avail_degrees
private

Used by branch_tour to track avoided edges.

Like fix_degrees, but for track BranchNode::Dir::Down edges in edge_stats. Initialized to all instance.node_count() - 1, and decremented. No tour can exist if an entry is less than two.

std::vector<int> CMR::ABC::Executor::fix_degrees
private

Used by branch_tour to track fixed edges.

An instance.node_count() length array initialized to all zero, and incremented for the ends of every BranchNode::Dir::Up edge in edge_stats. No tour can exist if any entry is greater than two.

util::SquareUT<char> CMR::ABC::Executor::inds_table
private

Used by branch_tour to track the edges in edge_stats.

An instance.node_count() dimensional upper triangular matrix with inds_table(i, j) being 'A' if the edge is to be avoided, 'W' if the edge is wanted (fixed to one), and '\0' otherwise.
