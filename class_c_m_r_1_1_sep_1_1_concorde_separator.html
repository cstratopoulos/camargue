Detailed Description

Abstract base class for calling Concorde separation routines.

Separator classes based on separation routines from Concorde should derive from this class and provide an implementation of find_cuts which calls an appropriate separation routine or sequence of routines. See FastBlossoms, BlockCombs, or SegmentCuts for examples.
